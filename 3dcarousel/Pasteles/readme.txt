Photo Dragger FX version 1.0.2.5232

Hello and thank you for using Photo Dragger FX. By downloading our FlashXML product you have become part of the FX community and we are very pleased to welcome you and wish you a long-lasting FlashXML experience.

The Photo Dragger FX's features are:
	- customizable widths and heights for the whole application
	- configurable reflection
	- the text is HTML/CSS formatted
	- many background and shade properties
	- customizable skin for the drag bar
	- multiple roll over/out effects
	- vertical / horizontal position
	- different customizable properties for images
	- many other features on the Live Demo

You can install our component on several platforms, from simple HTML sites to Flash projects. We provide installation instructions for multiple platforms, including video tutorials.
- for HTML: www.flashxml.net/photo-dragger.html#swmi-free-install
- for Flash: www.flashxml.net/photo-dragger.html#swmi-free-flash
- for WordPress: www.flashxml.net/photo-dragger.html#swmi-wordpress
- for Joomla: www.flashxml.net/photo-dragger.html#swmi-joomla
- for Facebook: www.flashxml.net/photo-dragger.html#swmi-facebook

By downloading our Photo Dragger FX you have access to one of the most customizable Flash component on the internet. All of our components are XML driven, so there is no need to use Adobe Flash to change the component's attributes or its assets.
The easiest way to start customizing our product is with its LiveDemo, which has all the component's attributes available.
To read more about how Flash components work with XML files, and get a good understanding of how our components work please visit this FAQ: www.flashxml.net/short-explanation-on-how-fx-components-read-xml-files.html
Most of our users want to use our component from a different folder than the location of the HTML page. To see how to embed a component that's in a different folder than your page please read www.flashxml.net/path-issues-with-swf-files-configured-via-xml.html
Since most of our users need to use special characters in their text fields we have created a special FAQ just for this: www.flashxml.net/special-characters-for-fx-components.html

Answers to the most common questions and problems of our users are also provided on the FAQ section from our website. You can find it here: www.flashxml.net/faq/.

This is a list with all the product's attributes and a short description about what each attribute does:
componentWidth - product's width
componentHeight - product's height
preloader - loading animation
position - text field's position
center - center images according to orientation
offset - images' position
useColorbox - use image pop-up effect on click
picWidth - images' width
picHeight - images' height
picDistance - distance between images
picCornerRadius - images' corners adjustment
rotationOffset - images' rotation angle
picBorderColor - images' border color
picBorderAlpha - images' border opacity 
picBorderWidth - images' border size
expandPercent - image's expand percent on mouse over
func - expand animation type
method - expand animation behavior
duration - expand animation duration
rollOverBehaviour - images effect on mouse over
brightnessOut - images' brightness effect
brightnessOver - brightness effect on mouse over
brightnessSpeed - brightness effect speed
useReflection - use images reflection
reflectionDistance - distance between images and reflection
reflectionAlpha - reflection opacity
reflectionSpread - reflection size
skin - drag bar skin set
dragEase - drag animation delay
dragBlur - drag animation blur effect intensity
delay - text field's appearance delay
distance - text field's vertical position
padding - text field's padding
cornerRadius - text field's corners adjustment
appearance - text field's appearance direction
easingFunction - text field's appearance animation
easingDuration - text field's appearance duration
fontFace - text's font type
fontSize - text's font size
fontColor - text's font color
backColor - text field's 1st background color
backColor2 - text field's 2nd background color
backAlpha - text field's background opacity
borderColor - text field's border color
borderThickness - text field's border size
useArrow - use arrow for text field
arrowPos - text field's arrow position
arrowSize - text field's arrow size
embedFonts - text smoothness
fillType - background type
backgroundImage - background image
backgroundFitting - background fitting
backgroundColor1 - background color when image background type not used
backgroundColor2 - background color when linear or radial background type used
backgroundAlpha - background opacity
bgCornerRadius - background corners' adjustments
gradientRotation - background colors rotation
gradientRatio - background colors adjustment
gradientPosX - background colors' horizontal position
gradientPosY - background colors' vertical position
useShade - use over mask
shadeColor - mask color
shadeAlpha - mask opacity
shadeRatio - mask adjustment
shadeWidth - mask width
shadeHeight - mask height
